DREAM RUN

About Dream Run

The 'Dream Run - Run for a Cause' event organized by the NSS Unit is a platform for participants to contribute to a noble cause while also enjoying the spirit of community engagement. Every year, the event brings together individuals to run for a social cause or to uplift the society as a whole. The marathon is not only a physical challenge but also an opportunity to create awareness about important social issues such as health, education, environmental sustainability, and more. The funds raised through the event go towards supporting various initiatives and programs aimed at addressing these issues. Apart from the physical challenge and the social impact, the 'Dream Run - Run for a Cause' event also fosters a sense of togetherness and community spirit. It provides an opportunity for participants to connect with like-minded individuals, build new relationships, and create lasting memories. So come join the 'Dream Run - Run for a Cause' event and make a difference in your community while experiencing the joy of running and community engagement.

RUN 2023

The NSS Unit's flagship event, "Dream Run - Run for a Cause," is an annual marathon that brings together individuals from all walks of life to contribute towards a social cause and uplift society. This year, we are proud to support children with autism and mental illnesses by partnering with the Ramchandra Kurulkar Bahuddeshiya Shikshan Sanstha. Our aim is to raise significant funds to support the nonprofit's efforts in improving the lives of children with disabilities and autism.

FREQUENTLY ASKED QUESTIONS.

What is Dream Run'23?

Participating in the Dream Run'23 for a cause like autism can be an excellent way to make a difference in the lives of children and families affected by this disorder.
